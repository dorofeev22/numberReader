Техническое задание.

Общее описание - программа читающая файл со списком чисел и выводящая на экран n максимальных из них.

Требования (условия):
- консольная программа
- читает файл большого размера
- в файле целые числа из диапазона от -2147483648 до 2147483647, каждое число на отдельной строке
- пользователь решает сколько маскимальных чисел выводить на экран и в каком порядке сортировать
